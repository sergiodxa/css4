¿Qué es CSS4?

Primero que todo CSS4 no existe como tal, esto es debido a que desde que salio CSS3 el estandar se dividió en módulo (por ejemplo el módulo background) y cada uno tiene su propia versión, habiendo módulos en la versión 3 (CSS3) y otros nuevos en la versión 1 (como FlexBox).

En esta guía hay información sobre nuevas características (nuevos módulos) y nuevas versiones de algunos módulos que se vienen en el futuro principalmente la versión 4 (por eso CSS4) del módulo de selectores.

Selectores (nivel 4)

:not(s1, ...sN)

Este nuevo selector permite estilizar elementos que no posean otros selectores como puede ser no tener una clase o no ser el primer elemento.
